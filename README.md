# Unity_GameViewCaptureButton

<img src="https://github.com/XJINE/Unity_GameViewCaptureButton/blob/main/Screenshot.png" width="100%" height="auto" />

Add a button on the left side of toolbar to capture Game View.

## Importing

You can use Package Manager or import it directly.

```
https://github.com/XJINE/Unity_TimeScaleEditorTool.git?path=Assets/Packages/GameViewCaptureButton
```

NOTE: Because of the limitation of UnityPackageManager, it needs to install dependencies first.

### Dependencies

This project use following resources.
